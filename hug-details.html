Create/Update Data Components

Using designs served via Zeplin, I developed all screens related to creating and updated hug program related data.

Reactive Forms and Validation were both required to make all PUT and POST API calls.

CRUD Operations and API Consumption

I was given the task of implementing all CRUD Operations and API consumption via the web portal using Angular 2's http module. All raw json data recieved from a Ruby on Rails backend was converted over to custom objects via an Angular 2 object model. This allowed for stronger typed code and less overall technical debt.
