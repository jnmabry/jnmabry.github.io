HUG - Help Us Grow

The HUG web application is a CMS used for a mobile app created to help children with low literacy rates. HUG is the most recent project that I have worked on during my time as an Interapt intern. It is built with all of the standard web technologies but more importantly utillizes Angular2 and TypeScript. It is loosely based off of another Interapt product called Togglit, so understanding the Togglit base code was imperative while laying out the app. I became alot more comfortable with concepts including routing, networking and the overall architecture of an Angular2 app during my time on this project. It was also for a great cause.

Tools Used:

HTML5/SCSS/JavaScript/Angular2/TypeScript

Weather.ly

Weather.ly was one of the first projects introduced during the web education portion of the TEKY1 program. It required calling two separate APIs using AJAX, and DOM manipulation using jQuery. The app requires the user to input a zipcode in order to retrieve the local weather for any given area. An API call is first made in order to retrieve the latitude and longitude for the given zipcode, then a second call is made to retrieve the weather using that information. The data recieved is then used to build an html template that is appended to the DOM and shown to the user via the apps UI.

Tools Used:

HTML5/CSS3/jQuery/AJAX/UI Design

Map.ly

Map.ly picked up where Weather.ly left off. This project had many of the same requirements as Weather.ly, but incorporated the Google Maps API instead.

Tools Used:

HTML5/CSS3/jQuery/AJAX/GOOGLE MAPS API/UI DESIGN
